`html2rss` is a community-driven project. Whether you are a user, a developer, or a self-hoster, there are many ways to get involved and help improve the project.

## Quick Links

- [**Project Roadmap**](https://github.com/orgs/html2rss/projects/3/views/1): View current work, plans, and priorities.
- [**GitHub Discussions**](https://github.com/orgs/html2rss/discussions): Ask questions, share tips, and connect with the community.
- [**Report an Issue**](https://github.com/html2rss/html2rss/issues): Report bugs or propose new features for the engine.
- [**Web App Issues**](https://github.com/html2rss/html2rss-web/issues): Report bugs for the web interface.

## Ways to Contribute

### 1. Create and Share Feed Configs

The most popular way to help! If a site you love doesn't have an RSS feed, create a config for it and share it in the [Feed Directory](/feed-directory/).

[**Learn how to contribute configs**](/get-involved/contributing/)

### 2. Improve the Documentation or Website

Found a typo? Want to improve a guide? This website is open source and we welcome your pull requests.

[**Documentation Source Code**](https://github.com/html2rss/html2rss.github.io)

### 3. Improve the Ruby Engine or Web App

If you are a Ruby developer, you can help improve the core logic or the web interface.

- [**html2rss (Ruby Gem)**](https://github.com/html2rss/html2rss)
- [**html2rss-web (Web App)**](https://github.com/html2rss/html2rss-web)

## Self-Hosting & Community

### Host a Public Instance

Running a reliable deployment benefits everyone. If you have a stable instance, consider adding it to the [community instance list](https://github.com/html2rss/html2rss-web/wiki/Instances).

### Help Others

Join the [Discussions](https://github.com/orgs/html2rss/discussions) and help new users get their first feeds running. Your experience is valuable to others!

`html2rss` is made possible by the support of our community. If you find the project useful, please consider sponsoring development.
