html2rss builds RSS feeds from websites which do not have one.

Start using RSS feeds right away or find out what html2rss has to offer.
View the feeds How it works

html2rss is a ecosystem of components with the goals to:
  1. generate RSS feeds by scraping websites (and JSON APIs).
  2. make it easy to build and use a feed config.
  3. have multiple decentralized and always up to date instances.
