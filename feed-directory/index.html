Welcome to the Feed Directory!

This directory contains a list of pre-built configurations to create RSS feeds for various websites.

What is an Instance URL?

An "Instance URL" is the address of a running `html2rss-web` application. You can use a publicly available instance or, for a more decentralized and robust web, you can host your own.

By hosting your own instance, you contribute to the resilience of the html2rss ecosystem and help keep the web open and accessible to everyone.
